Creare un nuovo progetto con nest:
- nest new nome-progetto

-------------------------------PRISMA------------------------------------------
Installare prisma (come dev dependency):
- npm i prisma -D

Inizializzare prisma (crea il file .env e la cartella prisma):
- npx prisma init

Aggiungere nel file ./prisma/schema.prisma le modifiche per il database e l'indirizzo del file del database

Lanciamo una migration SQL per modificare il database (se il file non esiste verr√† creato)
- npx prisma migrate dev --name nome_migration
(l'opzione dev salva gli statement SQL eseguiti sul db)

Se facciamo modifiche al file schema.prisma dobbiamo lanciare
- npx prisma generate (legge il file e modifica il Prisma client)
- npx prisma migrate dev --name nome_migration (lancia i comandi SQL sul database e ne modifica effettivamente la struttura)

------------------------------MODULO DB----------------------------------------
Creiamo un modulo db e service:
- nest g module database
- nest g service database

Altre cose da aggiungere, come la connessione al db nel database service
https://docs.nestjs.com/recipes/prisma#use-prisma-client-in-your-nestjs-services
